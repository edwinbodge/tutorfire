Project Overview

Students and Tutors Welcome

Tutorfire is a new way for students to connect with tutors that best meet their demands. The app has two modes, one for students and one for tutors. After creating an account and signing in, the user will be able to use the app.

Display Your Courses

In student mode, the user specify what subject they need help in and what times work best for them.

After querying a database of tutors, the matching list of tutors will be shown to the user, along with their contact information. The student will then be able to select a specific tutor from the list and start a chat with them, so they can set up meeting locations and times, as well as discuss the subject material.

Pick Your Times

While in tutor mode, the user can set the classes in which they are proficient and what time slots they are available. This information will be used to update the database of tutors. Tutors will also be able to see recent messages from students and schedule meetings with them.

Find Your Next Tutor

We use Firebase to handle messaging and user authentication services.

Students and Tutors have profiles that show their photo, their classes, as well as their average rating out of 5 stars.

Remote MySQL Database - Schema

Users(uid, name, photo_url)

CanTeach(uid, subject, course)

TimesFree(uid, day, start_time, end_time)

IsTutorOf(tutor_uid, student_uid, chat_hash)

TutorsStudentIn(tutor_uid, student_uid, subject, course)

RatesTutorAs(tutor_uid, student_uid, rating)

Example SQL statements

Suppose I am a tutor and want to now add CS 250 to the list of courses I can teach.

We would execute the following SQL statement.

INSERT INTO CanTeach (uid, subject, course)

VALUES (MY_UID, "CS", "250")

Suppose I am a student and want to get the list of all my current tutors and which courses they tutor me in.

SELECT IsTutorOf.tutor_uid, name, photo_url, chat_hash, rating, subject, course

FROM IsTutorOf, Users, TutorsStudentIn, RatesTutorAs

WHERE IsTutorOf.student_uid = STUDENT_UID AND uid = IsTutorOf.tutor_uid

AND IsTutorOf.student_uid = TutorsStudentIn.student_uid AND IsTutorOf.tutor_uid = TutorsStudentIn.tutor_uid

AND IsTutorOf.tutor_uid = RatesTutorAs.tutor_uid AND IsTutorOf.student_uid = RatesTutorAs.student_uid
