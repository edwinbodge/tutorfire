Harrison Lundberg

I created the web service on a Duke VM using MySQL and PHP. This web service stores all the information about tutors - which courses they teach and at what times.

I then started working on the WebServiceInterface package of our project, whose interface is methods that Student and Tutor mode activities can call to get information from the web service. These methods are implemented using HTTP requests with Volley. I also implemented the necessary JSON parsers (JSONParsers package) to translate the web server’s responses into data objects (DataClasses package). These data objects are then passed onto the appropriate activities in either Student or Tutor Mode.
